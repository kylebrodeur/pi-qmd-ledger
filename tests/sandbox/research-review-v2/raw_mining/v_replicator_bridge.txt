│ To achieve a "Star Trek replicator" capability where a natural language      │
│ command reliably produces a physical part, the system must bridge the gap    │
│ between the probabilistic nature of generative AI and the rigid safety       │
│ requirements of physical machinery. If left unchecked, pure Large Language   │
│ Models (LLMs) instructed to generate or optimize G-code will often produce   │
│ highly dangerous, non-conforming outputs. Because they suffer from           │
│ "token-minimization bias" and lack true geometric semantic understanding,    │
│ LLMs might dangerously delete critical safety commands (like G28             │
│ return-to-safe-position or G43 tool length compensation) or remove entire    │
│ pocket-milling operations simply to shorten the code length [1, 2].          │
│                                                                              │
│ World Models and Neuro-Symbolic validation layers solve this by dividing the │
│ process into two distinct phases: a cognitive planner and an absolute safety │
│ guardrail.                                                                   │
│                                                                              │
│ **1. The World Model (The Cognitive Planner)**                               │
│ Instead of mapping text directly to code, **World Models function as learned │
│ simulators that internalize the causal structures, thermodynamics, and       │
│ physical constraints of the factory floor** [3, 4]. Utilizing architectures  │
│ like Joint-Embedding Predictive Architectures (JEPA), the World Model        │
│ operates in a low-dimensional latent space to learn invariant structural     │
│ relationships, such as how cutting force impacts tool degradation [5, 6].    │
│ When given a natural language intent, **the World Model runs thousands of    │
│ "what-if" counterfactual simulations to spatial-reason and forecast the      │
│ physical consequences of a toolpath before it is executed** [4, 5]. In       │
│ hierarchical frameworks, the World Model decomposes the high-level           │
│ linguistic instruction into a sequence of compact, visually grounded         │
│ subgoals, providing explicit spatial constraints that safely guide the       │
│ low-level execution policy [7, 8].                                           │
│                                                                              │
│ **2. The Neuro-Symbolic Validation Layer (The Safety Guardrail)**            │
│ Once the World Model proposes an optimized G-code sequence or action plan,   │
│ it must pass through a Neuro-Symbolic (NeSy) architecture before reaching    │
│ the physical CNC machine. **This layer combines the adaptive generative      │
│ capabilities of the neural network with strict, deterministic symbolic logic │
│ to ensure absolute physical safety, auditability, and geometric compliance** │
│ [9-11].                                                                      │
│                                                                              │
│ Specifically, systems utilize **Neurosymbolic Attribute Grammars (NSGs)**,   │

│ symbolic engine explicitly checks boundaries—such as ensuring a proposed     │
│ feed rate does not exceed the spindle's power limit—and instantly blocks or  │
│ filters any generated code that violates these manufacturing constraints**   │
│ [13].                                                                        │
│                                                                              │

│ physical G-code execution is 100% syntactically correct, geometrically       │
│ compliant, and physically safe to run [11, 13].                              │
