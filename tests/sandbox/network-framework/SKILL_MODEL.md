# Skill Model

## Purpose
Define how knowledge becomes executable.

## Core Idea
A Skill is a compressed operational unit that can expand into:
- Teach
- Assist
- Do
- Simulate

## Structure
- Intent
- Inputs / Outputs
- Preconditions
- Constraints
- Failure Modes
- Success Criteria
- Nodes Required

## Expansion Modes
Teach → explanation
Assist → guided execution
Do → autonomous execution
Simulate → dry run

## Research Gaps
- Minimum viable compression
- Safety thresholds
- Skill evolution over time
